## Задача 3. Улучшение кода: найти и исправить проблемы
**Условие**. Даётся фрагмент кода (ты можешь взять свой или использовать пример ниже) с типичными проблемами: гонка данных, игнорирование ошибок, отсутствие валидации, неэффективные аллокации. Задача — исправить и добавить тесты.
Пример «плохого» кода:
```go
var cache = make(map[string]string)

func Get(key string) string {
 return cache[key] // нет проверки, гонка данных
}

func Set(key, value string) {
 cache[key] = value // гонка данных
}
```
